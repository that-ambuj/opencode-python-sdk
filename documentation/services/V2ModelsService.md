## v2_model_list

Retrieve available v2 models ordered by release date.

- HTTP Method: `GET`
- Endpoint: `/api/model`

**Parameters**

| Name     | Type                                                    | Required | Description |
| :------- | :------------------------------------------------------ | :------- | :---------- |
| location | [V2ModelListLocation](../models/V2ModelListLocation.md) | ❌       |             |

**Return Type**

`List[ModelV2Info]`

**Example Usage Code Snippet**

```python
from opencode_api import Opencode
from opencode_api.models import V2ModelListLocation

sdk = Opencode(
    access_token="YOUR_ACCESS_TOKEN",
    timeout=10000
)
location=V2ModelListLocation(
    directory="directory",
    workspace="workspace"
)

result = sdk.v2_models.v2_model_list(location=location)

print(result)
```
