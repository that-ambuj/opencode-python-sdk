| :----------- | :------------------------------------------------------------ | :------- | :---------------- |

| request_id   | str                                                           | ✅       |                   |
| directory    | str                                                           | ❌       |                   |
| workspace    | str                                                           | ❌       |                   |

**Return Type**

`bool`

**Example Usage Code Snippet**

```python
from opencode_api import Opencode
from opencode_api.models import PermissionReplyRequest

sdk = Opencode(
    access_token="YOUR_ACCESS_TOKEN",
    timeout=10000
)

request_body = PermissionReplyRequest(
    reply="once",
    message="message"
)

result = sdk.permission.permission_reply(
    request_body=request_body,
    request_id="per",
    directory="directory",
    workspace="workspace"
)

print(result)
```
