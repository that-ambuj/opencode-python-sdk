## project_update

Update project properties such as name, icon, and commands.

- HTTP Method: `PATCH`
- Endpoint: `/project/{projectID}`

**Parameters**

| Name         | Type                                                      | Required | Description       |
| :----------- | :-------------------------------------------------------- | :------- | :---------------- |
| request_body | [ProjectUpdateRequest](../models/ProjectUpdateRequest.md) | ❌       | The request body. |
| project_id   | str                                                       | ✅       |                   |
| directory    | str                                                       | ❌       |                   |
| workspace    | str                                                       | ❌       |                   |

**Return Type**

`Project`

**Example Usage Code Snippet**

```python
from opencode_api import Opencode
from opencode_api.models import ProjectUpdateRequest

sdk = Opencode(
    access_token="YOUR_ACCESS_TOKEN",
    timeout=10000
)

request_body = ProjectUpdateRequest(
    name="name",
    icon={
        "url": "url",
        "override": "override",
        "color": "color"
    },
    commands={
        "start": "start"
    }
)

result = sdk.project.project_update(
    request_body=request_body,
    project_id="projectID",
    directory="directory",
    workspace="workspace"
)

print(result)
```
