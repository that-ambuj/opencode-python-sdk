| Name         | Type                                                      | Required | Description       |
| :----------- | :-------------------------------------------------------- | :------- | :---------------- |
| request_body | [QuestionReplyRequest](../models/QuestionReplyRequest.md) | ❌       | The request body. |
| request_id   | str                                                       | ✅       |                   |
| directory    | str                                                       | ❌       |                   |
| workspace    | str                                                       | ❌       |                   |

**Return Type**

`bool`

**Example Usage Code Snippet**

```python
from opencode_api import Opencode
from opencode_api.models import QuestionReplyRequest

sdk = Opencode(
    access_token="YOUR_ACCESS_TOKEN",
    timeout=10000
)

request_body = QuestionReplyRequest(
    answers=[
        [
            "incupid"
        ]
    ]
)

result = sdk.question.question_reply(
    request_body=request_body,
    request_id="que",
    directory="directory",
    workspace="workspace"
)

print(result)
```
